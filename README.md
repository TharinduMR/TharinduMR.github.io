# TharinduMR.github.io

This repository hosts the source for my personal website built with GitHub Pages.

## About

This site contains personal projects, blog posts, and portfolio pages. It is implemented with HTML, CSS, and JavaScript, with a small amount of Python used for tooling.

Language composition (approx.):
- CSS: 56%
- JavaScript: 28%
- HTML: 14%
- Python: 2%

## Usage

To preview the site locally:

1. Clone the repository:

   git clone https://github.com/TharinduMR/TharinduMR.github.io.git

2. Serve the files with a simple static server. For example, using Python 3:

   cd TharinduMR.github.io
   python3 -m http.server 8000

3. Open http://localhost:8000 in your browser.
